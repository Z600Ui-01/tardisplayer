# 🌌 TARDIS Player (타디스 플레이어)

**TARDIS Player**는 닥터후(Doctor Who) 오디오 드라마 팬들을 위해 특수 제작된 **웹 기반 오디오 플레이어 및 지능형 자막 생성기**입니다. 

---

## ✨ 핵심 기능 (Key Features)

- 🎧 **오디오 트랙 자동 병합 재생**
  - 여러 개로 조각난 에피소드 파일들을 업로드하면 브라우저 내부에서 하나의 트랙으로 매끄럽게 이어 붙여 끊김 없이 재생합니다. (ID3 태그 앨범 아트 추출 지원)
- 📝 **닥터후 맞춤형 STT (Speech-to-Text)**
  - **OpenAI Whisper API**를 사용합니다. 
  - `TARDIS`, `Dalek`, `Gallifrey` 등 위스퍼가 헷갈리기 쉬운 닥터후 고유명사와 영국식 발음을 정확히 인식하도록 특수 프롬프트가 튜닝되어 있습니다.
- 🇰🇷 **캐릭터 관계 기반 AI 번역**
  - **Anthropic Claude (Sonnet) API**를 사용합니다.
  - 닥터와 컴패니언 사이의 반말/존댓말, 빌런의 오만한 말투, 영국식 유머 등 캐릭터의 관계와 성격을 반영하여 자연스러운 구어체로 번역합니다.
- 🙈 **스포일러 방지 모드 (Spoiler Prevention)**
  - 아직 재생되지 않은 미래의 자막들을 흐리게(Blur) 처리하여, 눈이 먼저 글을 읽어버려 오디오 드라마의 몰입이 깨지는 것을 방지합니다.
- 💾 **SRT 완벽 호환 및 화면 캡처**
  - 생성된 자막을 `.srt` 파일로 내보내거나, 기존 자막을 불러올 수 있습니다.
  - 맘에 드는 대사가 나올 때 `[CAPTURE]` 버튼을 눌러 플레이어 화면을 예쁜 이미지로 즉시 저장할 수 있습니다.

---

## 🛠️ 기술 스택 (Tech Stack)

- **Frontend:** Vanilla HTML, CSS, JavaScript (프레임워크 없이 순수 웹 기술로 구현)
- **APIs:** - OpenAI API (Whisper-1)
  - Anthropic API (Claude 3.5)
- **Libraries:**
  - `html2canvas` (화면 캡처용)
  - `jsmediatags` (오디오 앨범 아트 추출용)

## 🚀 사용 방법 (How to Use)

별도의 설치나 서버 구축이 필요 없습니다. 브라우저만 있으면 바로 작동합니다.

이 프로젝트는 원활한 API 통신과 브라우저 보안 정책(CORS) 준수를 위해 로컬 웹 서버 환경에서 실행하는 것을 권장합니다.

1. 이 저장소의 파일들을 다운로드하거나 클론합니다.
2. **VS Code**(Visual Studio Code)에서 프로젝트 폴더를 엽니다.
3. 확장 프로그램(Extensions)에서 **Live Server**를 설치합니다.
4. `tardis_player.html` 파일을 우클릭하고 **[Open with Live Server]**를 클릭하거나, VS Code 우측 하단의 **[Go Live]** 버튼을 누릅니다.
5. 우측 상단의 열쇠(`🔑`) 아이콘을 눌러 **OpenAI**와 **Anthropic** API 키를 입력합니다.
   *(입력한 키는 브라우저를 닫으면 사라지며, 서버로 전송되지 않고 오직 본인의 자막 생성에만 사용됩니다.)*
6. 좌측의 `[AUDIO]` 메뉴를 눌러 재생할 오디오 파일(들)을 선택합니다.
7. `[GENERATE]` 버튼을 누르면 타임 볼텍스를 통과하며 자막이 물질화됩니다!

> **⚠️ API 사용 시 주의사항**
> - OpenAI Whisper API의 정책상 한 번에 전송할 수 있는 파일 용량은 **25MB**로 제한됩니다. 
> - 플레이어 내부에서 자동으로 용량을 검사하여 경고를 띄워줍니다. 파일이 너무 크다면 20MB 이하로 분할하여 업로드해 주세요. (분할된 파일도 플레이어가 알아서 하나로 합쳐서 재생해 줍니다!)
